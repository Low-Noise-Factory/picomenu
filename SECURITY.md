# Security Policy

## Reporting a Vulnerability

We take security issues seriously. If you discover a security vulnerability in this project, please report it using GitHub's **Private vulnerability reporting** feature.

### How to Report

1. Go to the **Security** tab of this repository
2. Click on **Report a vulnerability**
3. Fill out the vulnerability details form
4. Submit your report

Your report will be reviewed promptly, and we'll work with you to understand and address the issue.

## What to Include

When reporting a vulnerability, please provide:

- A description of the vulnerability
- Steps to reproduce the issue
- Potential impact
- Any suggested fixes (if you have them)

Thank you for helping keep this project secure!
